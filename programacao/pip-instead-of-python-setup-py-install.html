PIP instead of python setup.py install

  • Thu 21 March 2013
  • programação
  • arruda
  • #pip, #python, #setup.py

:${IMAGE_TAG} A nice way to install any package that has it's setup.py file is to use the pip install -e command:

{lang="python"} pip install -e .

OBS: Use this in the same directory that has the setup.py file

This way you can have it nicely controlled by PIP =)
